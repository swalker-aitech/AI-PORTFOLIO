# Current Project

## Titan AI Document Assistant

Titan is a modular AI assistant designed to answer questions from user-provided documents using a controlled document-grounded approach.

## Capabilities

* Multi-document selection and management
* Document-based question answering
* Conversational memory
* Local LLM integration
* Modular Python architecture
* Strict document-grounded responses

## Technology Stack

* Python
* Ollama
* Mistral Large Language Model
* Git/GitHub
* LangChain concepts
* Prompt engineering

## Architecture

Titan follows a modular architecture:

User
↓
Application Interface
↓
Document Manager
↓
Document Loader
↓
AI Reasoning Layer
↓
Response Generation
↓
Conversation Memory
