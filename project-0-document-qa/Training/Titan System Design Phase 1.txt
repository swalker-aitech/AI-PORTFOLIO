Titan System Design Phase 1
From here forward (our operating model)

Every build will follow a simple pattern:

1. What are we building? (Requirement)

Example: “Save chat history”

2. What are the responsibilities? (Design)
load data
save data
handle missing file
3. Where does it live? (Structure)
memory.py
4. What are the functions? (Interface)
load_memory()
save_memory()
5. Code it
6. Test it
